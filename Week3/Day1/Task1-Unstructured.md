The project is about a website that converts code from a language to another. The homepage will have an option for the user to register/login, and a huge textbox for the user to enter code to convert without creating an account. There will also be a button for the user to click to start converting, and searchable dropdown menus for the user to be able to search for a programming language to convert from and another to convert to. It runs using a backend LLM API that takes the original code and converts it to another language based on the choice of the user. It will have 3 daily limit usage for each website visitor, or limits based on subscription tiers. Registering a free account grants the user up to 5 daily usages instead of the 3 without an account. The output will be copyable.
